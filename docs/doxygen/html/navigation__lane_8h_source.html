43 // A navigation path tuple.
44 //
45 // first element: original navigation line index of the current navigation path.
46 // A negative value indicates illegal.
47 //
48 // second element: half of the lateral distance to the left adjacent navigation
49 // path, that is, the left width of the lane generated by this navigation path.
50 // If the navigation path is generated based on lane markers, the value is the
51 // perceived left lane width. If there is no left adjacent navigation path, the
52 // value is "default_left_width_". A negative value indicates illegal.
53 //
54 // third element: half of the lateral distance to the right adjacent navigation
55 // path, that is, the right width of the lane generated by this navigation path.
56 // If the navigation path is generated based on lane markers, the value is the
57 // perceived right lane width. If there is no right adjacent navigation path,
58 // the value is "default_right_width_". A negative value indicates illegal.
59 //
60 // fourth element : a shared pointer of the current navigation path.
61 typedef std::tuple<int, double, double, std::shared_ptr<NavigationPath>>
62  NaviPathTuple;

71 // pair.second: the distance between the vehicle's initial position and the
